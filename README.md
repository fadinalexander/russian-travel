# Проект: Путешествие по России

### Описание
Проект является одностраничным сайтом с реализацией адаптивной верстки 
для разных разрешений экранов.

### Технологии
В проекте используются технологии:
* grid;
* flex-box;
* файловая структура по БЭМ (Nested);
